# Restaurant Order Booking System - Frontend

This is the frontend application for the Restaurant Management System, built with React, Vite, and Tailwind CSS.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Backend server running on `http://localhost:3000`

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```
2. Create environment file:
   ```bash
   cp .env.example .env
   ```
3. Update `.env` if necessary:
   ```env
   VITE_API_URL=http://localhost:3000
   VITE_SOCKET_URL=http://localhost:3000
   ```

### Development
Run the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:3002`.

## 🛠️ Tech Stack
- **React 18**
- **Vite**
- **Tailwind CSS**
- **Radix UI**
- **Socket.io Client**
